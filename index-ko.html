자주 묻는 질문

1. 내 데이터는 여러 기기에서 동기화할 수 있나요?
네, 여러 기기에서 동기화 기능을 제공합니다.

2. 구독에는 어떤 기능이 포함되나요?
버전 (1.0.0)에서는 다음과 같은 차별화된 기능을 제공합니다:

  • 노트북과 자산의 생성 제한이 향상되었습니다.
  • 영수증 OCR 인식을 통해 자동 회계를 지원합니다.
  • 모든 광고가 제거되었습니다.
  • ...

3. 구독이 만료되면 생성한 저축 계획은 어떻게 되나요?
저희는 사용자의 저축 계획을 보존하지만, 구독을 갱신하기 전까지는 표시되지 않습니다.

4. 구독은 환불이 가능한가요?
모든 구독은 App Store의 환불 정책을 따릅니다.
